## IStateMachineService 需要的方法：
```csharp
Task<Result> TransitionToAsync(ExecutionCard card, ExecutionCardStatus newStatus);
bool CanTransition(ExecutionCard card, ExecutionCardStatus newStatus);
```

## ISubmitService 需要的方法：
```csharp
Task<Result<ExecutionRun>> SubmitAsync(int cardId, int runId, string evidence, string executorId);
Task<Result<SpecEvaluation>> EvaluateSpecAsync(int specId, int runId);
```

## 代码风格要求
- 接口继承 `IBaseService<T>`
- 实现继承 `ServiceBase<T, IRepository<T>>`
- 实现 `IDependency` 接口让 Autofac 自动注册
- 遵循 partial 类模式：生成的代码放在主文件，自定义业务放在 Partial 目录
