# E-Kanban 开发任务：2.1 实现 Azure Boards API 客户端

## 任务说明
按照 E-Kanban 开发计划，第二批次第一个任务：实现 Azure Boards API 客户端，用于从 Azure Boards 拉取工作项。

## 前置条件
- ✅ 第一批次所有任务已完成（实体、仓储、服务）

## 开发规范
$(cat docs/volcore-development-guide.md)

## 需求
$(cat docs/requirements-structured.md)

## 位置
- 放在 `EKanban/AzureBoards/` 目录
- 接口：`IAzureBoardsClient.cs`
- 实现：`AzureBoardsClient.cs`

## 功能要求
1. 配置：从 `appsettings.json` 读取配置
   - OrganizationUrl
   - Project
   - PersonalAccessToken
   
2. 需要实现的方法：
   ```csharp
   // 获取所有工作项
   Task<List<AzureWorkItem>> GetAllWorkItemsAsync();
   
   // 获取单个工作项详情
   Task<AzureWorkItem> GetWorkItemAsync(int workItemId);
   
   // 添加评论到工作项
   Task AddCommentAsync(int workItemId, string comment);
   
   // 更新工作项状态
   Task UpdateStateAsync(int workItemId, string state);
   ```

3. 使用 HttpClient 调用 Azure DevOps REST API
   - 使用 `IHttpClientFactory` 创建 HttpClient
   - 添加 PAT 认证
   - 处理分页响应
   - 错误处理和日志

## 数据结构
- `AzureWorkItem` 类需要包含：
  - Id
  - Title
  - Description
  - State
  - AreaPath
  - IterationPath
  - CreatedDate
  - UpdatedDate
  - CreatedBy
  - ChangedBy
