## 功能要求

### ICopilotCliClient - 底层 CLI 调用
```csharp
// 调用 copilot CLI，返回输出
Task<CopilotCliResult> ExecuteAsync(string prompt);
```
- 使用 `System.Diagnostics.Process` 启动 `copilot` 命令
- 传递 `copilot -p "prompt" --allow-all-tools` 参数
- 捕获标准输出和标准错误
- 等待执行完成
- 返回退出码、输出内容、错误信息
- 处理超时（默认 30 分钟超时）

配置从 `appsettings.json` 读取：
- `CopilotCliPath` - copilot 可执行文件路径（默认 `copilot`）
- `CommandTimeoutMinutes` - 命令超时时间（默认 30）

### IAiExecutionService - AI 执行高层逻辑
```csharp
// 执行 AI 任务
Task<Result> ExecuteAiTaskAsync(ExecutionCard card);
```
- 根据卡片信息和 Spec 构造 prompt
- 调用 `ICopilotCliClient` 执行
- 获取输出后，自动调用 Submit API 提交结果
- 处理执行错误，记录到 ExecutionRun

Prompt 构造示例：
```
你正在执行 E-Kanban 系统调度的 AI 开发任务。

任务卡片：{{card.Title}}
任务描述：{{card.Description}}

验收标准 (Spec):
{{spec.Content}}

请按照 Spec 要求完成任务，修改代码后提交结果。
```
