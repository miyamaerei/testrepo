### 2. 执行迁移

**SQL Server 迁移脚本**：

```sql
-- 添加IsManualCreated字段
ALTER TABLE ExecutionCards
ADD IsManualCreated BIT NOT NULL DEFAULT 0;

-- 创建索引（可选，提高查询性能）
CREATE INDEX IX_ExecutionCards_IsManualCreated
ON ExecutionCards(IsManualCreated);
```

**执行步骤**：
1. 登录数据库管理工具（如SSMS）
2. 连接到目标数据库
3. 执行上述SQL脚本
4. 验证迁移结果，确保字段添加成功
