## 功能概述

本方案旨在实现 Copilot 执行路径与卡片绑定项目的关联，确保 AI 执行时能够在正确的项目目录中工作。主要包括：

- 检查卡片绑定的项目信息
- 根据项目信息动态设置 Copilot 执行的工作目录
- 确保执行环境与项目路径正确关联
- 验证执行结果与项目的一致性
