## 验收标准

### 1. 功能验收

1. **看板卡片展示**：
   - 卡片按状态分组展示（新建、就绪、执行中、已完成、失败）
   - 每个卡片显示正确的标题、描述、状态等信息
   - 卡片数量显示正确

2. **同步功能**：
   - 点击 "同步 Azure Boards" 按钮后，显示同步成功提示
   - 同步后页面数据更新

3. **详情查看功能**：
   - 点击卡片上的 "查看详情" 按钮后，显示卡片详情对话框
   - 对话框包含基本信息、阶段进度和文件变更
   - 阶段进度按顺序显示，支持展开/收起日志
   - 文件变更列表显示正确的文件路径、变更类型等信息

4. **执行触发功能**：
   - 点击卡片上的 "执行" 按钮后，显示执行成功提示
   - 卡片状态更新为 "执行中"

5. **重新执行触发功能**：
   - 点击卡片上的 "重新执行" 按钮后，显示重新执行成功提示
   - 卡片状态更新为 "就绪"
