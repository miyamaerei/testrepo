## 功能概述

本方案旨在实现手动创建看板的功能，与Azure Boards同步的看板明确区分，并提供完整的管理功能。主要包括：

- 手动创建看板卡片
- 与Azure同步看板的区分标识
- 后端字段添加与API接口实现
- 前端独立管理组件开发
- 完整的增删改查功能支持
